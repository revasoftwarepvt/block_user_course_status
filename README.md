# My Course Progress (block_user_course_status)

## Description
**My Course Progress** is a Moodle block plugin that displays the learning progress of the logged-in student across all their enrolled courses.

The block provides a quick and clear overview of course completion status, helping students easily track their learning progress from the dashboard.

## Features
- Displays all courses enrolled by the logged-in student
- Shows current course status:
  - Not Started
  - In Progress
  - Completed
- Displays completed module count vs total module count
- Shows overall course completion percentage
- Student-specific view (only shows the logged-in user's courses)
- Clean and simple UI for better readability

## Use Case
This block is ideal for:
- Students who want a quick overview of their learning progress
- Institutions that want to improve learner engagement
- Dashboards that need progress tracking at a glance
